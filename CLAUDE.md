# Murse Fun Research Project Site
This is the website for my personal mini projects that I find interesting. 

## Design Principles
1. The bakcground of the blog should be some dark color, with the light font color to create a sense of technology.
2. The organization of the main page should be a title "Murse Fun Research" followed by boxes showing each blog entry.
3. The blog entry box should show the title, date and cover image of the blog, with a liquid glass effect.
4. The main page should look good on different screen width, containing one column of boxed on shorter screen and two columns on wider screen.
5. The blog source will be written in markdown, with each blog in a seperate folder with markdown file and other images. The website should be able to render the markdown to a good webpage.
6. The blog page should be one column on no matter the screen width.

## File Structure
1. All the .html file should be put in the `html` folder (except the index.html file).
2. All the .css file should be put in the `css` folder.
3. All the javascript file should be put in the `js` folder.
4. Do not use heavy weight framework like React or Vue for building the site.
5. All the blogs are put in the `blog` folder, labeled by numbers like 1, 2, 3, etc. Inside each blog folder (like `blog/1/`), there should be cover image `cover.jpg` or `cover.png`, a file containing the time and title `meta.json`, and the main blog `blog.md` along with other resources.

## Other Notices
1. Do not edit ant other file except the files under this project.
2. When you finish building the website, create 3 test blogs to help me verify the correctness of the design.
